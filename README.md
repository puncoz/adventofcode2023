# AdventOfCode 2023

Problem: [https://adventofcode.com/2023](https://adventofcode.com/2023)

## Instruction to run

1. Clone the repository
```shell
git clone https://github.com/puncoz/adventofcode2023.git
```

2. Install the `yarn` dependencies
```shell
yarn
```

3. Run the solution for day
```shell
yarn solve <day-num>

// eg:
yarn solve 1
```
